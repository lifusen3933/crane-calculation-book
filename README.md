# 🏗️ 起重机设计计算书自动生成系统

基于模板分析的起重机设计计算书自动生成程序，能够根据输入的设计参数自动进行工程计算并生成完整的设计计算书文档。

## 📋 功能特点

- ✅ **自动化计算**: 根据输入参数自动完成所有工程计算
- ✅ **标准规范**: 严格遵循GB/T 3811-2008《起重机设计规范》
- ✅ **全面分析**: 涵盖载荷、结构、机构、稳定性等12个主要章节
- ✅ **结果验证**: 自动检查设计是否满足规范要求
- ✅ **报告生成**: 自动生成完整的设计计算书（Markdown格式）
- ✅ **多种界面**: 支持命令行和Web两种操作界面

## 📐 计算内容

### 12个主要章节

1. **载荷和载荷组合的分析与计算**
2. **主要受力结构件的强度、刚性**
3. **连接的设计计算**
4. **整机抗倾覆稳定性**
5. **起升机构部件设计或选型计算**
6. **小车运行机构部件设计或选型计算**
7. **起重机运行机构部件设计或选型计算**
8. **电气系统设计或选型计算**
9. **液压系统设计或选型计算**
10. **安全保护装置设计或选型计算**
11. **设计使用年限**
12. **其他设计计算**

## 🚀 快速开始

### 环境要求

- Python 3.9+
- pip 或 conda

### 安装步骤

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **运行程序**
   ```bash
   # 命令行界面
   python main.py cli

   # Web界面
   python main.py web
   ```

### 使用示例

#### 命令行界面

```bash
$ python main.py cli
```

#### Web界面

```bash
$ python main.py web
```

然后在浏览器中打开 `http://localhost:8501`
